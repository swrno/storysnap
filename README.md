# StorySnap
I want to build a app for papa, he has the hobby of capturing historical places and write about that in story format. 

- Login, Signup option for users
- Anyone can post the story with images
- The post should go to admin panel, after approval that will be live on the feed.
- Should have support of translation to bengali
- Should have audio support for blogs, which will enable users to listen the writings.

Tech Stack: Next.js + Typescript, MongoDB, Cloudinary, Firebase Auth
